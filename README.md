# Discord 재고 관리 봇

이 Discord 봇은 SQLite 데이터베이스를 사용하여 가상 아이템의 재고와 가격을 관리하도록 설계되었습니다. 프로그램이 업데이트되거나 재시작되어도 데이터가 유지되도록 합니다.

## 필수 구성 요소

- Python 3.6 이상
- discord.py 라이브러리
- SQLite3 (Python에 포함)
